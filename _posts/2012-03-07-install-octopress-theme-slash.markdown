还是对于octopress折腾的记录，年纪大了记不住那么多的命令。

这个slash主题就是我现在用的这个，很简洁大方，同时还很喜欢它的灰色调。

{% codeblock %}
cd octopress
git clone git@github.com:tommy351/Octopress-Theme-Slash.git .themes/slash
rake install['slash']
rake generate
{% endcodeblock %}
